# Agentic RAG on Kubeflow (Expansion of kubeflow/docs-agent)

## ğŸ¯ Objective

Design and deploy a scalable, modular **Agentic RAG (Retrieval-Augmented Generation)** system that:

* Accepts natural language queries from users
* Routes queries through specialized agents
* Retrieves and merges relevant information from authoritative sources ("Golden Data")
* Generates structured, cited answers using an LLM served on Kubeflow via KServe
* Continuously ingests and updates knowledge using Kubeflow Pipelines
